Real-Estate Website || View Github Repository

In this project, I have built a location-based real estate website. It is a full stack web application with Django powering the backend and React Js the frontend.
I built the frontend very fast by using Material UI which is a very easy and intuitive React library for building UI components. I then built the API with Django Rest Framework (DRF) and I made requests to the API with React. I added a geographic aspect to this project by placing the property listings on a map. I achieveD that with React-Leaflet which provides a binding between React and Leaflet library.
